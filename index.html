SVG: gráficos vetoriais escaláveis
  • imagem criada a partir de código XML ou HTML totalmente gerada por codigo html
  • Facil dimencionamento e modificação de imagem utilizando formas atraves do atributo d que contem uma serie de caminhos que definem o caminho a ser desenhado
  • As imagnes SVG fazem parte do DOm do navegador e por isso podem ser definidos utilizado JS ou CSS
Comandos do atribudo d
  • M
  • L
  • H
  • V
  • C
  • S
  • Q
  • T
  • A
  • Z

Letra minuscula definine um caminho relativo e letra maiuscula define um caminho absoluto

Dasharray e dashoffset
  • Dasharray: Controla o padrão de traços e espaços usados
  • Dashoffset: define o local onde o traço de a stroke começará

    • Esses dois atributos juntos podem ser utilizados para animar caminhos SVG usando CSS

    Sintaxe

    stroke-dasharray: none | <dasharray>> | inherit;

    stroke-dashoffset: none
